# 乳房標記12點鐘圖
使用React 建置 12點鐘乳房標記模組。可記錄標記"座標"、"圓心距離"、"大小"。
使用左鍵增加腫瘤，左鍵點選腫瘤修改大小、刪除。

## Installation
With npm installed, run  
1. ```$ npm install```  
2. ```$ npm start```
