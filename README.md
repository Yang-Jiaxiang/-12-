# 乳房標記12點鐘圖
使用React 建置 12點鐘乳房標記模組。可記錄標記"座標"、"圓心距離"、"大小"。

## View
![944129ba-9853-44cf-8390-74d19dd175f9](https://user-images.githubusercontent.com/81738019/213499920-7f7dd924-fa21-46f3-8f51-67b40e7012e1.gif)

## Installation
With npm installed, run  
1. ```$ npm install```  
2. ```$ npm start```
